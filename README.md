# The Sprak Programming Language Project

Sprak is an implementation of Thornston Ball [Monkey Programming Language](https://monkeylang.org/) in [Odin](https://odin-lang.org/). It closely follows Thornston Ball's [books](https://interpreterbook.com/).

## Prerequisite

If you want to follow along, you need a copy of Thornston Ball's Writing an Interpreter in Go.

You also need to install the [Odin compiler](https://odin-lang.org/docs/install/).
